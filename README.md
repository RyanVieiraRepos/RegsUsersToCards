# Sistema de Cadastro de Usuários

Sistema de cadastro e listagem de usuários com HTML, CSS e JavaScript.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Fetch API

## Funcionalidades

- Formulário de cadastro com validação
- Integração com API REST (POST/GET)
- Listagem de usuários em cards
